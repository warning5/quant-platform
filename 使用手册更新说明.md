#### 2.1 位置

- 包含详细的指标说明和使用建议

#### 2.2 核心内容

**衰减原理说明**：
- 解释因子预测能力随时间衰减的原理
- 说明通过计算滞后IC值来观察衰减趋势

**核心指标详解**：
- **因子有效期（期数）**：IC绝对值首次低于0.02的期数，表示因子预测能力的持续时间
- **半衰期（期数）**：IC降至初始值50%所需的期数，反映衰减速度
- **衰减系数（λ）**：指数衰减模型的系数，值越大衰减越快
- **拟合优度R²**：衰减模型的解释程度，越接近1表示拟合越可靠

**图表说明**：
- IC绝对值曲线：蓝色折线展示衰减趋势
- IC值曲线：橙色折线展示带符号的IC变化
- 阈值虚线：IC=0.02处的有效期限标记

**使用建议（Alert提示框）**：
- 调仓频率选择：衰减快的因子需要频繁调仓
- 策略设计：有效期长的因子适合长期持有
- 因子组合：组合不同衰减特性的因子可以提高稳定性
- 持仓周期：根据半衰期确定合理的持仓周期

**访问路径**：
- 详细的四步操作指南
- 注意事项：旧版本报告需要重新运行检测

### 3. 因子相关性分析章节

#### 3.1 位置
- 作为独立的大章节添加（Title level 3）
- 位于"因子衰减分析"和"因子策略"之间
- 使用 PartitionOutlined 图标标识

#### 3.2 核心内容

**相关性等级标准（Card卡片）**：
- 强相关（|r| ≥ 0.7）：因子高度重叠，建议避免同时使用
- 中等相关（0.4 ≤ |r| < 0.7）：因子有一定重叠，需谨慎组合
- 弱相关（0.2 ≤ |r| < 0.4）：因子有一定独立性，可以组合
- 无相关（|r| < 0.2）：因子独立，适合组合使用

**热力图解读（Card卡片）**：
- 红色区域：正相关，表示因子同向变化
- 蓝色区域：负相关，表示因子反向变化
- 白色区域：无相关或弱相关，因子相互独立
- 对角线：永远为1，表示因子与自身的相关性

**使用方法（Card卡片）**：
- 步骤1：选择因子（至少2个）
- 步骤2：设置日期范围
- 步骤3：计算相关性
- 步骤4：分析结果

**相关性数据表（Card卡片）**：
- 因子对：参与分析的因子1和因子2
- 相关系数：Pearson相关系数值，用颜色标记强度
- 相关性强度：基于相关系数划分的等级
- 样本数：用于计算相关性的数据点数量

**应用场景（Alert提示框）**：
- 提高策略稳定性：选择相关性低的因子组合
- 避免信息冗余：避免同时使用相关性超过0.7的因子
- 分散风险：不同类型的因子通常相关性较低
- 因子筛选：优先选择与其他因子相关性低的因子
- 因子去重：发现高度相关的因子时保留表现更好的

**访问路径**：
- 四步操作指南
- 强调这是独立的功能模块
