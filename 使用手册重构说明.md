## 技术改进

### 1. 导航菜单
- 使用 Ant Design Menu 组件的 children 属性实现层级结构
- 使用 inlineCollapsed 属性实现收起功能
- 使用 useState 管理收起状态
- 使用 scrollToSection 实现平滑滚动

### 2. 响应式设计
- 使用 xs={0} md={...} 实现移动端隐藏菜单
- 使用 xs={24} md={...} 实现响应式内容布局
- 使用 gutter={[16, 16]} 优化间距

### 3. 视觉优化
- 使用不同颜色的 Card 区分内容类型
- 使用 emoji 和图标提升视觉效果
- 使用 type="inner" 创建内嵌卡片
- 使用 hoverable 属性添加悬停效果
