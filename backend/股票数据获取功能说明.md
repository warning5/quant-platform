# 股票基础数据获取功能说明

## 概述

本功能用于通过 AKTools API 获取股票基础数据，包括股票列表、实时行情、历史数据等，并存储到 MySQL 数据库中。

## 依赖项

- Java 21
- Spring Boot 3
- MySQL 数据库
- AKTools 服务（需单独部署）

## 数据库配置

### MySQL 配置 (application-mysql.yml)

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/stock?serverTimezone=Asia/Shanghai&useUnicode=true&characterEncoding=utf8&useSSL=false&allowPublicKeyRetrieval=true&tinyInt1isBit=false
    username: root
    password: 123456
```

### AKTools 配置

```yaml
quant:
  aktools:
    enabled: false  # 设置为true启用AKTools数据获取
    domain: http://127.0.0.1:9090
    base-url: http://127.0.0.1:9090/api/public/
```

### AKTools 服务

AKTools 是基于 AKShare 的 Python 数据接口服务，需要单独部署。

**启动 AKTools 服务**:
```bash
pip install aktools
aktools
# 默认监听 http://127.0.0.1:9090
```

## 功能列表

### 1. 获取A股股票列表

**接口**: `GET /api/stock/list`

**功能**: 获取A股所有股票的基本信息列表

**响应示例**:
```json
[
  {
    "代码": "000001",
    "名称": "平安银行",
    "市场": "SZ"
  },
  {
    "代码": "600000",
    "名称": "浦发银行",
    "市场": "SH"
  }
]
```

### 2. 获取个股实时行情

**接口**: `GET /api/stock/spot/{symbol}`

**参数**:
- `symbol`: 股票代码（如：000001.SZ）

**响应示例**:
```json
{
  "最新价": 12.34,
  "涨跌幅": 1.23,
  "涨跌额": 0.15,
  "成交量": 123456789,
  "成交额": 1234567890,
  "最高": 12.50,
  "最低": 12.10,
  "今开": 12.20,
  "昨收": 12.19
}
```

### 3. 获取个股详细信息

**接口**: `GET /api/stock/info/{symbol}`

**参数**:
- `symbol`: 股票代码

**响应示例**:
```json
{
  "股票名称": "平安银行",
  "总市值": 234567890000,
  "流通市值": 123456789000,
  "所属行业": "银行",
  "总股本": 19405918198,
  "流通股本": 19405918198,
  "市净率": 0.85,
  "市盈率": 6.32
}
```

### 4. 获取个股历史行情

**接口**: `GET /api/stock/history/{symbol}`

**参数**:
- `symbol`: 股票代码
- `startDate`: 开始日期（yyyy-MM-dd）
- `endDate`: 结束日期（yyyy-MM-dd）

**响应示例**:
```json
[
  {
    "日期": "2024-01-01",
    "开盘": 12.30,
    "收盘": 12.45,
    "最高": 12.50,
    "最低": 12.20,
    "成交量": 123456789,
    "成交额": 1234567890,
    "涨跌幅": 1.22,
    "涨跌额": 0.15,
    "换手率": 2.35
  }
]
```

### 5. 同步股票列表

**接口**: `POST /api/stock/sync/list`

**功能**: 从AKTools同步A股股票列表数据到本地数据库

### 6. 同步个股历史数据

**接口**: `POST /api/stock/sync/history/{symbol}`

**参数**:
- `symbol`: 股票代码
- `days`: 历史天数（默认365）

**功能**: 同步指定股票的历史行情数据到数据库

### 7. 批量同步历史数据

**接口**: `POST /api/stock/sync/batch-history`

**参数**:
- `symbols`: 股票代码列表（JSON数组）
- `days`: 历史天数（默认365）

**功能**: 批量同步多只股票的历史行情数据

## 使用示例

### 1. 获取股票列表

```bash
curl -X GET "http://localhost:8080/api/stock/list"
```

### 2. 获取个股行情

```bash
curl -X GET "http://localhost:8080/api/stock/spot/000001.SZ"
```

### 3. 同步历史数据

```bash
curl -X POST "http://localhost:8080/api/stock/sync/history/000001.SZ?days=30"
```

### 4. 批量同步

```bash
curl -X POST "http://localhost:8080/api/stock/sync/batch-history?days=30" \
  -H "Content-Type: application/json" \
  -d '["000001.SZ", "600000.SH", "000002.SZ"]'
```

## 支持的股票代码格式

- 深市: `000001.SZ`, `300001.SZ`
- 沪市: `600000.SH`, `688001.SH`

## 注意事项

1. AKTools 服务需要单独部署和启动
2. 使用前请将 `application.yml` 中的 `aktools.enabled` 设置为 `true`
3. 批量同步时注意API调用频率限制
4. 历史数据同步会覆盖数据库中已有数据
