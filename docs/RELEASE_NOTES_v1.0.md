## 🎉 概述

Quant Platform v1.0.0 是项目的首个正式 Release。它标志着平台的核心功能链路已经完整可用——从数据采集到因子计算、策略回测、智能推荐、AI 分析、模拟交易，形成闭环。

本版本面向**量化学习者和研究者**，提供完整的开源代码和文档，可用于学习量化系统的全链路设计。

---

## ✨ 核心功能清单

### 因子引擎
- 35+ 内置因子，覆盖 6 大类（动量 / 价值 / 质量 / 情绪 / 资金 / 形态）
- IC 分析：IC 序列、ICIR、RankIC、分层回测、多空收益
- 衰减分析：有效期、半衰期、衰减系数 λ、拟合优度 R²
- 因子健康监控：自动衰减检测 → 降级 → 复活，每日 22:00 巡检
- 相关性分析：60 日滚动相关矩阵 + Union-Find 聚类去冗余
- 权重优化：等权 / Markowitz 有效前沿 / 风险平价
- 因子全生命周期：DRAFT → TESTING → ACTIVE → DEGRADED → DEPRECATED

### 策略引擎
- 14 套 ACTIVE 策略（含多因子选股、动量、均值回归、形态识别、自定义脚本）
- 置信度系统 P0-P3：冷启动保护，小样本自动降权
- Groovy 脚本支持自定义策略信号

### 回测引擎
- 事件驱动模拟：日频调仓，手续费 + 滑点
- 绩效指标：年化收益、最大回撤、Sharpe、Sortino、Calmar、IR、胜率
- Brinson 归因：配置效应 + 选股效应 + 交互效应
- Fama-French 3 因子分析
- Monte Carlo 模拟：Bootstrap 200-1000 次，VaR / CVaR
- Walk-Forward 优化：滚动窗口验证，防过拟合
- 参数网格搜索：Sharpe / 年化 / Calmar 三目标

### 智能推荐（P1-P6 管线）
- P1: IC 筛选（|IC| < 0.015 剔除）
- P2: 半衰期评估
- P3: 拥挤度预警
- P4: 季频校正
- P5: 多周期一致性验证
- P6: ICW 融合（上限 35%，反转有害因子）

### AI 智能分析
- DeepSeek 大模型驱动
- 个股因子画像 + 财务诊断 + 风险预警
- 多维交叉验证（估值 / 成长 / 质量 / 动量）

### 组合风控
- 跨策略个股去重
- SW2 行业集中度硬上限
- 回撤监控 + Regime 市场状态自适应
- 模拟盘交易：信号 → 执行 → 风控 → 预警完整闭环

### 数据工程
- 5 源采集：Baostock / 腾讯 / AKShare / 东方财富 / 同花顺
- ClickHouse 列存加速（可选 MySQL 替代）
- 前复权自动刷新
- 定时任务依赖链编排

### 前端界面
- Dashboard 总览
- 因子管理（列表 / IC 分析 / 健康监控 / 相关性 / 权重优化）
- 策略管理（列表 / 详情 / 编辑器）
- 回测分析（创建 / 报告 / 对比 / 蒙特卡洛 / 参数优化）
- AI 分析
- 选股器
- 模拟盘监控
- 市场温度计 / 个股分析
- 行情数据 / 财务数据 / 交易日历
- 使用手册（内置）

## ⚠️ 已知限制

1. **数据源依赖**：Baostock 匿名登录按 IP 限流，高频采集可能触发封禁。建议分批采集（50 只/批，间隔 10 分钟）。
2. **前复权快照性**：前复权价格是时点快照，每次除权除息会追溯更新历史因子。除权后需重刷历史数据。
3. **北交所数据**：北交所数据通过腾讯接口获取，PE/PB 仅快照值非历史序列。
4. **模拟盘**：模拟盘为纸面交易，不连接真实券商，不执行真实买卖。
5. **AI 分析**：DeepSeek 分析结果仅供参考，不构成投资建议。
6. **商用限制**：腾讯证券接口商用需授权，开源版仅供个人学习研究使用。
