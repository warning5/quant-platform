### 3. 因子选股-打分策略（基于您提供的Python实现）

#### 核心逻辑
采用估值因子打分模型进行量化价值投资筛选。

#### 打分规则
```
因子1 (PE_TTM): < 33 → 10分, 否则0分
因子2 (PS_TTM): < 2.01 → 10分, 2.01-10.01 → 5分, 否则0分
因子3 (PB): < 1.01 → 10分, 1.01-3.01 → 5分, 否则0分
综合得分 = 因子1 + 因子2 + 因子3
选择得分最高的前30只股票
```

#### 实现优势
- ✅ 数据需求全部具备（PE_TTM、PS_TTM、PB已实现）
- ✅ 计算逻辑简单直接（条件判断+加法）
- ✅ 可直接转换您的Python代码

```java
@Component
public class MultiFactorRegressionAnalyzer {
    
    public RegressionResult analyzeFactors(
            List<StockData> stocks, 
            List<FactorData> factors) {
        // 1. 数据预处理
        Matrix factorMatrix = buildFactorMatrix(factors);
        Vector returns = buildReturnVector(stocks);
        
        // 2. 执行多元线性回归
        OLSMultipleLinearRegression regression = new OLSMultipleLinearRegression();
        regression.newSampleData(returns.toArray(), factorMatrix.getData());
        
        // 3. 计算统计指标
        double[] coefficients = regression.estimateRegressionParameters();
        double rSquared = regression.calculateRSquared();
        
        return RegressionResult.builder()
                .coefficients(coefficients)
                .rSquared(rSquared)
                .build();
    }
}
```

**任务2：因子贡献度计算**
```java
@Service
public class FactorContributionCalculator {
    
    public Map<String, Double> calculateContributions(
            RegressionResult regressionResult,
            List<FactorData> factors) {
        
        Map<String, Double> contributions = new HashMap<>();
        double[] coefficients = regressionResult.getCoefficients();
        
        for (int i = 0; i < factors.size(); i++) {
            String factorName = factors.get(i).getName();
            double contribution = coefficients[i + 1] * factors.get(i).getStdDev();
            contributions.put(factorName, contribution);
        }
        
        return contributions;
    }
}
```

**任务5：因子打分策略实现**

```java
@Component
public class FactorScoringStrategyImpl implements FactorScoringStrategy {
    
    @Override
    public List<Stock> selectStocks() {
        List<Stock> candidates = getAllStocks();
        List<StockScore> scores = new ArrayList<>();
        
        for (Stock stock : candidates) {
            double peScore = calculatePEScore(stock.getPeTTM());
            double psScore = calculatePSScore(stock.getPsTTM());
            double pbScore = calculatePBScore(stock.getPb());
            
            double totalScore = peScore + psScore + pbScore;
            scores.add(new StockScore(stock, totalScore));
        }
        
        // 按得分排序，取前30名
        return scores.stream()
                .sorted((a, b) -> Double.compare(b.getScore(), a.getScore()))
                .limit(30)
                .map(StockScore::getStock)
                .collect(Collectors.toList());
    }
}
```
