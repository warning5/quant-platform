## 📈 策略详细说明

### 1. Breakaway跳空突破策略

#### 核心思想
价格跳空突破关键阻力位或支撑位，形成强势突破信号。

#### 形态识别规则
```
跳空缺口 = 今日开盘价 - 昨日收盘价
向上突破: 缺口 > 0 且价格站上关键阻力位
向下突破: 缺口 < 例: -2% 且价格跌破关键支撑位
关键位识别: 前高/前低、整数关口、移动平均线等
```

#### 交易信号
- **买入信号**: 向上跳空突破关键阻力位
- **卖出信号**: 向下跳空突破关键支撑位
- **确认信号**: 突破后成交量放大

#### 实现状态
- ✅ 数据需求: OHLC数据（TechnicalFactorData已支持）
- ✅ 计算逻辑: 条件判断+缺口计算
- ✅ 调仓频率: 日频

### 2. 三浪下跌形态策略

#### 核心思想
价格呈现三段式下跌结构，每段下跌后都有反弹但高点逐级降低。

#### 形态识别规则
```
第一浪: 从高点开始的下跌（幅度较大）
第二浪: 反弹但未创新高（幅度较小）
第三浪: 再次下跌并创新低（通常幅度最大）
验证条件: 
- 三浪时间关系：第3浪 ≥ 第1浪时间
- 三浪幅度关系：第3浪 ≥ 第1浪幅度
- 波峰递减：第2浪高点 < 第1浪高点
```

#### 交易信号
- **做空信号**: 第三浪确认后（跌破第2浪低点）
- **止损设置**: 第3浪高点上方
- **目标位**: 第1浪幅度的1.618倍

#### 实现状态
- ✅ 数据需求: OHLC数据（TechnicalFactorData已支持）
- ✅ 计算逻辑: 波峰波谷识别算法
- ⚠️ 实现难度: 中等（需要峰值检测）

**任务1：EMA计算器实现**
```java
@Component
public class EMACalculator {
    
    public List<Double> calculateEMA(List<Double> prices, int period) {
        if (prices == null || prices.size() < period) {
            throw new IllegalArgumentException("数据不足");
        }
        
        List<Double> emaValues = new ArrayList<>();
        double multiplier = 2.0 / (period + 1);
        
        // 初始化：使用简单移动平均
        double initialEMA = calculateSMA(prices.subList(0, period));
        emaValues.add(initialEMA);
        
        // 递推计算后续EMA值
        for (int i = period; i < prices.size(); i++) {
            double ema = (prices.get(i) - emaValues.get(emaValues.size() - 1)) 
                        * multiplier + emaValues.get(emaValues.size() - 1);
            emaValues.add(ema);
        }
        
        return emaValues;
    }
    
    private double calculateSMA(List<Double> values) {
        return values.stream().mapToDouble(Double::doubleValue).average().orElse(0.0);
    }
}
```

**任务2：缺口识别工具**
```java
@Component
public class GapDetector {
    
    public List<Gap> detectGaps(List<TechnicalFactorData> dataList) {
        List<Gap> gaps = new ArrayList<>();
        
        for (int i = 1; i < dataList.size(); i++) {
            TechnicalFactorData today = dataList.get(i);
            TechnicalFactorData yesterday = dataList.get(i - 1);
            
            double gap = today.getOpenPrice() - yesterday.getClosePrice();
            double gapPercent = gap / yesterday.getClosePrice() * 100;
            
            // 定义显著缺口阈值（可根据需要调整）
            if (Math.abs(gapPercent) >= 1.0) {  // 1%以上的缺口
                Gap gapObj = Gap.builder()
                        .date(today.getDate())
                        .gapAmount(gap)
                        .gapPercent(gapPercent)
                        .direction(gap > 0 ? GapDirection.UP : GapDirection.DOWN)
                        .build();
                gaps.add(gapObj);
            }
        }
        
        return gaps;
    }
    
    public enum GapDirection {
        UP, DOWN
    }
}
```

**任务3：波峰波谷检测算法**
```java
@Component
public class PeakValleyDetector {
    
    public WaveStructure detectWaves(List<TechnicalFactorData> dataList, int windowSize) {
        List<Point> peaks = new ArrayList<>();
        List<Point> valleys = new ArrayList<>();
        
        // 检测波峰
        for (int i = windowSize; i < dataList.size() - windowSize; i++) {
            if (isPeak(dataList, i, windowSize)) {
                peaks.add(new Point(dataList.get(i).getDate(), 
                                  dataList.get(i).getHighPrice()));
            }
        }
        
        // 检测波谷
        for (int i = windowSize; i < dataList.size() - windowSize; i++) {
            if (isValley(dataList, i, windowSize)) {
                valleys.add(new Point(dataList.get(i).getDate(), 
                                    dataList.get(i).getLowPrice()));
            }
        }
        
        return WaveStructure.builder()
                .peaks(peaks)
                .valleys(valleys)
                .build();
    }
    
    private boolean isPeak(List<TechnicalFactorData> data, int index, int window) {
        double currentValue = data.get(index).getHighPrice();
        for (int i = Math.max(0, index - window); i <= Math.min(data.size() - 1, index + window); i++) {
            if (i != index && data.get(i).getHighPrice() >= currentValue) {
                return false;
            }
        }
        return true;
    }
    
    private boolean isValley(List<TechnicalFactorData> data, int index, int window) {
        double currentValue = data.get(index).getLowPrice();
        for (int i = Math.max(0, index - window); i <= Math.min(data.size() - 1, index + window); i++) {
            if (i != index && data.get(i).getLowPrice() <= currentValue) {
                return false;
            }
        }
        return true;
    }
}
```

**任务4：跳空突破策略实现**
```java
@Component
public class BreakawayPatternStrategy implements PatternRecognitionStrategy {
    
    @Autowired
    private GapDetector gapDetector;
    
    @Autowired
    private SupportResistanceFinder srFinder;
    
    @Override
    public boolean detectPattern(List<TechnicalFactorData> dataList) {
        List<Gap> gaps = gapDetector.detectGaps(dataList);
        if (gaps.isEmpty()) return false;
        
        Gap latestGap = gaps.get(gaps.size() - 1);\        TechnicalFactorData latestData = dataList.get(dataList.size() - 1);
        
        // 检查是否为有效突破
        List<Double> supportLevels = srFinder.findSupportLevels(dataList);
        List<Double> resistanceLevels = srFinder.findResistanceLevels(dataList);
        
        if (latestGap.getDirection() == GapDirection.UP) {
            // 向上突破：缺口 > 0 且价格站上关键阻力位
            return latestGap.getGapPercent() > 1.0 && 
                   isAboveResistance(latestData.getClosePrice(), resistanceLevels);
        } else {
            // 向下突破：缺口 < -2% 且价格跌破关键支撑位
            return latestGap.getGapPercent() < -2.0 && 
                   isBelowSupport(latestData.getClosePrice(), supportLevels);
        }
    }
    
    @Override
    public TradeSignal generateSignal(List<TechnicalFactorData> dataList) {
        if (!detectPattern(dataList)) {
            return TradeSignal.HOLD;
        }
        
        Gap latestGap = gapDetector.detectGaps(dataList).get(0);
        return latestGap.getDirection() == GapDirection.UP ? 
               TradeSignal.BUY : TradeSignal.SELL;
    }
}
```

**任务5：三浪下跌形态策略实现**
```java
@Component
public class ThreeWavePatternStrategy implements PatternRecognitionStrategy {
    
    @Autowired
    private PeakValleyDetector waveDetector;
    
    @Override
    public boolean detectPattern(List<TechnicalFactorData> dataList) {
        WaveStructure waves = waveDetector.detectWaves(dataList, 3);
        
        if (waves.getPeaks().size() < 2 || waves.getValleys().size() < 3) {
            return false;
        }
        
        List<Point> peaks = waves.getPeaks();
        List<Point> valleys = waves.getValleys();
        
        // 验证三浪结构
        return validateThreeWaveStructure(peaks, valleys);
    }
    
    private boolean validateThreeWaveStructure(List<Point> peaks, List<Point> valleys) {
        if (peaks.size() < 2 || valleys.size() < 3) return false;
        
        // 三浪下跌的时间和幅度关系验证
        Point firstWaveHigh = peaks.get(0);
        Point secondWaveHigh = peaks.get(1);
        Point firstWaveLow = valleys.get(0);
        Point secondWaveLow = valleys.get(1);
        Point thirdWaveLow = valleys.get(2);
        
        // 波峰递减验证
        boolean peakDecreasing = secondWaveHigh.getValue() < firstWaveHigh.getValue();
        
        // 第三浪幅度验证（通常大于等于第一浪）
        double firstWaveAmplitude = firstWaveHigh.getValue() - firstWaveLow.getValue();
        double thirdWaveAmplitude = secondWaveHigh.getValue() - thirdWaveLow.getValue();
        boolean amplitudeValid = thirdWaveAmplitude >= firstWaveAmplitude * 0.8;
        
        return peakDecreasing && amplitudeValid;
    }
    
    @Override
    public TradeSignal generateSignal(List<TechnicalFactorData> dataList) {
        if (!detectPattern(dataList)) {
            return TradeSignal.HOLD;
        }
        
        // 第三浪确认后的做空信号
        WaveStructure waves = waveDetector.detectWaves(dataList, 3);
        Point thirdWaveLow = waves.getValleys().get(2);\        TechnicalFactorData latestData = dataList.get(dataList.size() - 1);
        
        // 破坏第三浪低点时发出做空信号
        if (latestData.getLowPrice() < thirdWaveLow.getValue()) {
            return TradeSignal.SHORT;
        }
        
        return TradeSignal.HOLD;
    }
}
```

```java
@Component
public class PatternRiskManager {
    
    public RiskAssessment assessPatternRisk(List<TechnicalFactorData> dataList, 
                                          PatternType patternType) {
        // 1. 成交量验证
        boolean volumeConfirmed = validateVolume(dataList);
        
        // 2. 时间窗口检查
        boolean timeValid = validateTimeWindow(dataList, patternType);
        
        // 3. 技术面确认
        boolean technicalConfirmed = validateTechnicalSignals(dataList);
        
        // 4. 风险评级
        RiskLevel riskLevel = calculateRiskLevel(volumeConfirmed, timeValid, technicalConfirmed);
        
        return RiskAssessment.builder()
                .volumeConfirmed(volumeConfirmed)
                .timeValid(timeValid)
                .technicalConfirmed(technicalConfirmed)
                .riskLevel(riskLevel)
                .build();
    }
    
    private boolean validateVolume(List<TechnicalFactorData> dataList) {
        if (dataList.size() < 2) return false;
        
        TechnicalFactorData latest = dataList.get(dataList.size() - 1);
        TechnicalFactorData previous = dataList.get(dataList.size() - 2);
        
        // 突破时成交量应该放大
        return latest.getVolume() > previous.getVolume() * 1.5;
    }
    
    private RiskLevel calculateRiskLevel(boolean volumeOk, boolean timeOk, boolean techOk) {
        int score = (volumeOk ? 1 : 0) + (timeOk ? 1 : 0) + (techOk ? 1 : 0);
        switch (score) {
            case 3: return RiskLevel.LOW;
            case 2: return RiskLevel.MEDIUM;
            default: return RiskLevel.HIGH;
        }
    }
}
```

## 📊 策略应用建议

### 风险控制
- 单笔交易风险 ≤ 2%
- 形态确认后才入场
- 设置严格止损（形态高点/低点外1-2%）
- 成交量必须配合验证

### 组合应用
- 与MACD/SMA等趋势指标结合使用
- 在日线级别确认，在30分钟级别执行
- 成交量配合验证形态有效性
- 多时间框架共振确认

### 参数优化建议
- 跳空突破：缺口阈值1-3%可调
- 三浪结构：时间窗口3-7个交易日
- 波峰波谷：检测窗口期5-10根K线
- 支撑阻力：回看周期20-60个交易日

## ⚠️ 注意事项

1. **形态失效风险**：市场环境变化可能导致经典形态失效
2. **假突破问题**：需要多重确认避免假信号
3. **执行延迟**：形态确认往往滞后于最佳入场点
4. **市场适应性**：不同市场阶段形态有效性差异较大
5. **过度拟合风险**：参数优化时要注意样本外测试
