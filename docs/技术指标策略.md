## 📈 策略详细说明

### 1. MACD交易策略

#### 核心思想
利用MACD指标的金叉死叉信号进行趋势交易，结合柱状图变化判断趋势强度。

#### 计算公式
```
EMA12: 12日指数移动平均
EMA26: 26日指数移动平均  
MACD = EMA12 - EMA26
Signal = 9日EMA(MACD)
Histogram = MACD - Signal
```

#### 交易信号
- **买入**: MACD上穿Signal线且Histogram由负转正
- **卖出**: MACD下穿Signal线且Histogram由正转负
- **趋势确认**: Histogram持续放大表示趋势加强

#### 实现状态
- ✅ 数据需求：收盘价（TechnicalFactorData已支持）
- ✅ 计算基础：需新增EMA计算器
- ✅ 调仓频率：日频

### 2. SAR交易策略

#### 核心思想
抛物线转向指标，用于跟踪趋势并识别反转点。

#### 计算公式
```
AF = 加速因子 (0.02→0.2)
EP = 极值点 (最高价/最低价)
SAR = 昨日SAR + AF × (EP - 昨日SAR)
```

#### 交易信号
- **买入**: SAR从上方下穿价格
- **卖出**: SAR从下方上穿价格
- **趋势跟踪**: SAR在价格下方=上升趋势，上方=下降趋势

#### 实现状态
- ✅ 数据需求：OHLC数据（TechnicalFactorData已支持）
- ✅ 计算特点：需要极值点跟踪算法
- ✅ 调仓频率：日频

### 3. 加速震荡策略 (AC)

#### 核心思想
Bill Williams提出，通过AC指标判断趋势加速情况，比AO更敏感。

#### 计算公式
```
MP = (High + Low) / 2
AO = SMA(MP, 5) - SMA(MP, 20)
AC = AO - SMA(AO, 5)
```

#### 交易信号
- **买入**: AC值自下而上穿越零轴
- **卖出**: AC值自上而下下穿零轴
- **趋势强度**: AC值远离零轴表示趋势强劲

#### 实现状态
- ✅ 数据需求：OHLC数据（TechnicalFactorData已支持）
- ✅ 计算基础：复用现有SMA计算逻辑
- ✅ 调仓频率：日频

### 4. PSY交易策略

#### 核心思想
心理线指标，统计N日内上涨天数比例判断市场情绪。

#### 计算公式
```
PSY = N日内上涨天数 / N × 100%
标准参数：N=12日
```

#### 交易信号
- **买入**: PSY < 25%（超卖区域）
- **卖出**: PSY > 75%（超买区域）
- **中性**: 25% ≤ PSY ≤ 75%

#### 实现状态
- ✅ 数据需求：收盘价序列（TechnicalFactorData已支持）
- ✅ 计算基础：简单计数逻辑
- ✅ 调仓频率：日频

### 5. SMA移动均线策略

#### 核心思想
简单移动平均线交叉策略，识别趋势转换。

#### 计算公式
```
SMA(n) = ΣP_i / n （i=1到n）
常用组合：SMA(5)/SMA(20)交叉
```

#### 交易信号
- **买入**: 短期SMA上穿长期SMA（金叉）
- **卖出**: 短期SMA下穿长期SMA（死叉）
- **趋势确认**: 价格在SMA上方=多头趋势，下方=空头趋势

#### 实现状态
- ✅ 数据需求：收盘价（TechnicalFactorData已支持）
- ✅ 计算基础：MovingAverageCrossoverCalculator已实现
- ✅ 调仓频率：日频

### 策略工厂模式
```java
@Component
public class TechnicalStrategyFactory {
    
    @Autowired
    private List<TechnicalStrategy> strategies;
    
    public TechnicalStrategy getStrategy(String name) {
        return strategies.stream()
            .filter(s -> s.getStrategyName().equalsIgnoreCase(name))
            .findFirst()
            .orElse(null);
    }
}
```
