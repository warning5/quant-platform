### 1. 彼得·林奇PEG策略

#### 核心思想
寻找成长性与估值相匹配的优质股票，PEG值越低越好。

#### 筛选条件
```
- PEG < 1.0（理想值）
- 市盈率 10-20倍之间
- 盈利增长率 > 15%
- ROE > 15%
- 负债率 < 50%
```

#### 实现要点
- **数据需求**: PE、增长率、ROE、负债率
- **计算方法**: PEG = PE / (盈利增长率 × 100)
- **调仓频率**: 季度调仓
- **风险控制**: 设置止损线和最大持仓限制

#### Java实现示例
```java
@Component
public class LinchPEGStrategy implements InvestmentStrategy {
    
    @Autowired
    private ValuationFactorService valuationService;
    
    @Autowired
    private QualityFactorService qualityService;
    
    public List<Stock> selectStocks() {
        // 获取基础股票池
        List<Stock> candidates = getAllStocks();
        
        return candidates.stream()
            .filter(stock -> calculatePEG(stock) < 1.0)
            .filter(stock -> stock.getPe() >= 10 && stock.getPe() <= 20)
            .filter(stock -> getGrowthRate(stock) > 0.15)
            .filter(stock -> qualityService.calculateROE(stock) > 0.15)
            .filter(stock -> qualityService.calculateDebtRatio(stock) < 0.5)
            .sorted(Comparator.comparing(this::calculatePEG))
            .limit(30)
            .collect(Collectors.toList());
    }
    
    private double calculatePEG(Stock stock) {
        double peg = stock.getPe() / (getGrowthRate(stock) * 100);
        return Double.isFinite(peg) ? peg : 999;
    }
}
```

### 2. 巴菲特Alpha策略

#### 核心思想
投资具有强大竞争优势的优质企业，长期持有分享企业成长。

#### 筛选条件
```
- ROE > 20%（持续3年以上）
- 毛利率 > 40%
- 负债率 < 30%
- 经营现金流 > 净利润
- 市值 > 100亿
```

#### 实现要点
- **数据需求**: ROE历史数据、毛利率、现金流、市值
- **质量评估**: 连续多年财务指标稳定性
- **护城河识别**: 通过ROE分解分析竞争优势
- **长期视角**: 至少3-5年持有期

#### Java实现示例
```java
@Component
public class BuffettAlphaStrategy implements InvestmentStrategy {
    
    public List<Stock> selectQualityStocks() {
        List<Stock> candidates = getAllStocks();
        
        return candidates.stream()
            .filter(this::hasStrongMoat)
            .filter(this::hasConsistentROE)
            .filter(this::hasHighMargins)
            .filter(this::hasHealthyBalanceSheet)
            .filter(this::hasStrongCashFlow)
            .filter(stock -> stock.getMarketCap() > 100_000_000_000L)
            .sorted(Comparator.comparing(this::calculateQualityScore).reversed())
            .limit(20)
            .collect(Collectors.toList());
    }
    
    private boolean hasConsistentROE(Stock stock) {
        // 检查过去3年ROE是否都大于20%
        List<Double> roeHistory = getROEHistory(stock, 3);
        return roeHistory.stream().allMatch(roe -> roe > 0.20);
    }
}
```

### 3. 奥肖内西价值策略

#### 核心思想
低估值+高股息+低波动的价值投资组合。

#### 筛选条件
```
- 市净率 < 1.5
- 股息率 > 3%
- 3年波动率 < 30%
- 盈利稳定性 > 80%
- 行业分散度 > 5个行业
```

#### 实现要点
- **多因子融合**: 估值、分红、波动率三维筛选
- **风险分散**: 行业和个股双重分散
- **收益增强**: 通过股息率提升整体收益

#### Java实现示例
```java
@Component
public class OShaughnessyValueStrategy implements InvestmentStrategy {
    
    public Portfolio constructPortfolio() {
        List<Stock> valueStocks = selectValueStocks();
        Map<String, Double> weights = calculateWeights(valueStocks);
        
        return Portfolio.builder()
            .stocks(valueStocks)
            .weights(weights)
            .riskControl(RiskControl.builder()
                .maxSectorWeight(0.2)
                .maxIndividualWeight(0.05)
                .targetVolatility(0.15)
                .build())
            .build();
    }
}
```

### 4. 查尔斯·布兰德策略

#### 核心思想
逆向投资，在市场悲观时买入被低估的优质资产。

#### 筛选条件
```
- 市盈率历史分位数 < 20%
- 市净率历史分位数 < 20%
- 机构持仓下降 > 20%
- 技术面超卖（RSI < 30）
- 基本面良好（ROE > 10%）
```

#### 实现要点
- **时机把握**: 结合市场情绪和技术指标
- **安全边际**: 严重低估时才考虑买入
- **耐心等待**: 等待市场认知修复

### 5. 本杰明·格雷厄姆策略

#### 核心思想
寻找价格远低于内在价值的安全边际投资机会。

#### 筛选条件
```
- 市净率 < 1.2
- 市盈率 < 9
- 负债/净资产 < 0.5
- 流动比率 > 2.0
- 连续5年盈利
```

#### 实现要点
- **安全边际**: 价格至少比估算价值低30%
- **财务稳健**: 严格的资产负债表要求
- **分散投资**: 同时持有10-15只股票

### 6. 史蒂夫·路佛价值策略

#### 核心思想
结合价值投资和质量投资，寻找低估值的高质量公司。

#### 筛选条件
```
- EV/EBITDA < 8
- ROE > 15%
- ROA > 10%
- 负债率 < 40%
- 自由现金流 > 0
- 市值 > 50亿
```

#### 实现要点
- **估值合理**: EV/EBITDA多重估值方法
- **质量优先**: ROE、ROA等质量指标筛选
- **现金流导向**: 重视企业现金流创造能力

### 7. 三一投资管理策略

#### 核心思想
多维度价值筛选，综合PE、PB、股息率三个维度。

#### 筛选条件
```
- 市盈率最低的前400公司
- 市净率最低的前400公司  
- 股息率最高的前400公司
- 取交集后选择前30只
- 个股止损：低于成本价8%
- 大盘保护：5日下跌13%清仓
```

#### 实现要点
- **多因子交集**: 三个维度同时满足
- **数量控制**: 严格控制持仓数量
- **风控严格**: 双重止损机制

#### Java实现示例
``java
@Component
public class TrinityInvestmentStrategy implements InvestmentStrategy {
    
    private static final int SELECTED_COUNT = 400;
    private static final int PORTFOLIO_SIZE = 30;
    
    public List<Stock> selectStocks() {
        List<Stock> allStocks = getAllStocks();
        
        // 三个维度筛选
        List<Stock> peStocks = sortByPE(allStocks).limit(SELECTED_COUNT).collect(Collectors.toList());
        List<Stock> pbStocks = sortByPB(allStocks).limit(SELECTED_COUNT).collect(Collectors.toList());
        List<Stock> dyStocks = sortByDividendYield(allStocks).limit(SELECTED_COUNT).collect(Collectors.toList());
        
        // 取交集
        Set<Stock> intersection = new HashSet<>(peStocks);
        intersection.retainAll(pbStocks);
        intersection.retainAll(dyStocks);
        
        return intersection.stream()
            .limit(PORTFOLIO_SIZE)
            .collect(Collectors.toList());
    }
}
```

### 8. 迈克尔·普莱斯策略

#### 核心思想
寻找严重低估的价值股，重点关注资产负债表健康度。

#### 筛选条件
```
- 市净率 < 2.0
- 负债比例低于市场平均值
- 股东权益比率高于平均水平
- 市值 > 10亿
- 前30只股票入选
```

#### 实现要点
- **低估识别**: 市净率严格约束
- **财务健康**: 负债水平横向比较
- **规模适中**: 避免过小公司的流动性风险

## 📈 回测与实盘建议

### 历史回测
- **时间跨度**: 至少5年历史数据
- **交易成本**: 考虑佣金和滑点
- **分红处理**: 准确计算股息收益
- **复权处理**: 使用前复权价格

### 实盘部署
- **分批建仓**: 避免一次性满仓
- **持续监控**: 每日跟踪持仓表现
- **定期评估**: 月度策略效果评估
- **适时调整**: 根据市场环境微调参数
